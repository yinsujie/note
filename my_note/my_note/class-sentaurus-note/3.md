# 2-5-网格划分的基础知识与评价标准

### **delaunay三角网/三角化:**

满足所有内角和最大的优化准则为delaunay三角网。这种剖分方法遵循“最小角最大”和“空外接圆”准则，两者是等效的。

空外接圆准则表示当前三角形所形成的外接圆不包含其他的点。
