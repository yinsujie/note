### **单载流子仿真：**

```scheme{.line-numbers}
仿真MOSFET时，电子起主导作用，耦合求解电子和泊松与（电子、泊松、空穴）相差不大，因此求解（电子和泊松）会有更好的求解速度和收敛性，但空穴的准费米势是常数
```

### **反向倒推空穴准费米势：**

```scheme{.line-numbers}
RecomputeQFP after the solution of Coupled Poisson Electron update Efp
```

### **雪崩击穿：**

```scheme{.line-numbers}
不能用单载流子仿真，会有大的误差
```

### **Dopingwell：**

*.plt文件中Electrode Charge的意义
每个DopingWel中的准费米势可以被控制和扫描

```scheme{.line-numbers}
Plot{
DopingWells
}
```
